This plugin helps Wordpress pages to display Lemonwhale video content

== Description ==

This plugin was created by Lemonwhale to give customers an easy to use tool embed video content on Wordpress sites.

= Requirements =

- Wordpress version >= 3.0
- PHP version >= 5.2
- Modern web browser

== Installation ==

1. Install the plugin via the WordPress.org plugin directory.
